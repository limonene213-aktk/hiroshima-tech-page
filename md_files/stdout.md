# 標準出力ってなに？ 🖥️

Linuxでは、コマンドを実行すると結果が画面に表示されます。この結果のことを「**標準出力**」と言います！  
ここでは、標準出力について分かりやすく説明し、パイプ（`|`）を使った便利な操作例も紹介します。

---

## **1. 標準出力とは？**

- **標準出力（stdout）**は、コマンドが出した結果を画面に表示する仕組みのことです。
  - **例**：以下のコマンドを実行すると、「Hello, World!」というテキストが画面に表示されます。
    ```bash
    echo "Hello, World!"
    ```
    **結果**：
    ```
    Hello, World!
    ```

- **ポイント**：
  - 標準出力は「テキスト形式」で表現されます。
  - テキストなので、後から他のコマンドで処理したり、ファイルに保存したりできます。

---

## **2. パイプ（`|`）を使った標準出力の活用**

### **パイプの基本**
パイプ（`|`）は、**コマンドの標準出力を次のコマンドの入力として渡す**道具です。

#### **例：`ls`コマンドの結果を`grep`で絞り込む**
```bash
ls | grep "txt"
```
- **何をする？**
  - `ls`コマンドで現在のディレクトリにあるファイル一覧を表示。
  - パイプでその結果を`grep`に渡し、「txt」という文字列を含むファイルだけを抽出します。

---

## **3. 標準出力を処理するコマンドの例**

### **例1：`echo`と`grep`の組み合わせ**
```bash
echo -e "apple\nbanana\ncherry" | grep "a"
```
- **何をする？**
  - `echo`で「apple」「banana」「cherry」という3つの単語を標準出力。
  - パイプでその結果を`grep`に渡し、「a」を含む単語だけを表示。

**結果**：
```
apple
banana
```

---

### **例2：`ls`と`less`の組み合わせ**
```bash
ls | less
```
- **何をする？**
  - `ls`コマンドで現在のディレクトリにあるファイル一覧を表示。
  - パイプでその結果を`less`に渡し、スクロールして確認できるようにします。

---

### **例3：`ip a`と`grep`の組み合わせ**
```bash
ip a | grep "inet"
```
- **何をする？**
  - `ip a`コマンドでネットワーク情報を標準出力。
  - パイプでその結果を`grep`に渡し、「inet」という文字列を含む行だけを表示。

---

### **例4：`cat`と`tail`の組み合わせ**
```bash
cat /var/log/syslog | tail -n 5
```
- **何をする？**
  - `cat`で`/var/log/syslog`ファイルの内容を標準出力。
  - パイプでその結果を`tail`に渡し、最後の5行だけを表示。

---

## **4. 標準出力をファイルに保存する**

標準出力をファイルに保存したい場合は、**リダイレクト（`>`）**を使います。

### **例：`ls`の結果をファイルに保存**
```bash
ls > file_list.txt
```
- **何をする？**
  - `ls`コマンドの結果（標準出力）を`file_list.txt`というファイルに保存します。
  - **注意**：すでに存在する`file_list.txt`は上書きされます。

---

## **5. 練習してみよう！**

以下の手順で練習してみましょう：

1. **標準出力を確認**
   ```bash
   echo "This is standard output"
   ```
   **結果**：
   ```
   This is standard output
   ```

2. **パイプで処理**
   ```bash
   echo -e "red\ngreen\nblue" | grep "e"
   ```
   **結果**：
   ```
   green
   blue
   ```

3. **ファイル一覧を検索**
   ```bash
   ls | grep "txt"
   ```

4. **標準出力を保存**
   ```bash
   ls > output.txt
   cat output.txt
   ```

---

## **6. よく使うコマンドと標準出力の組み合わせ例**

| **コマンド**           | **何をする？**                              |
|------------------------|--------------------------------------------|
| `echo "text" | grep "t"` | "text"の中で"t"を含む文字列を表示する      |
| `ls | less`            | ファイル一覧をスクロールして確認する         |
| `ip a | grep "inet"`   | IPアドレスを抽出する                       |
| `cat file.txt | tail`  | ファイルの最後の部分だけ表示する            |
